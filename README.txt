Application should: 
1. Present a window to the user. 
2. Allow the user to upload two files: one paper and one memo (they must be in PDF format)
3. Once the user clicks okay, it should show a warning box to make sure that the paper and answer key correspond. 
4. Upon clicking okay, the application should break down the paper and the memo, taking screenshots of each question and answer. 
5. The application should then map each question image to each answer image and place their file locations row by row in a sql database, with other row entries left blank for users to input data on the marks awarded and comments for each question.
